Image Search Abstraction Layer --A simple API service

function:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Some Example:

	https://saintpeter-image-search.herokuapp.com/search/funny%20cats
	https://saintpeter-image-search.herokuapp.com/search/funny%20cats?offset=10
	https://saintpeter-image-search.herokuapp.com/latest
